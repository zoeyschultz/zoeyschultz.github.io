Introduction

Adam Ondra is widely considered the best climber in the world — but how does his performance actually compare in competitions and on real rock?

In this post, I use data from his IFSC results and outdoor ascents to explore his strengths, tradeoffs, and one surprising weakness that cost him at the Olympics.

Code 
rankings <- read_csv("data/adam_master_rankings.csv", show_col_types = FALSE)

filtered <- rankings |>
  filter(Discipline %in% c("lead", "boulder"),
         Date >= 2010, Date <= 2024)

avg_rankings <- filtered |>
  group_by(Date, Discipline) |>
  summarise(AverageRank = mean(Rank, na.rm = TRUE), .groups = "drop")

chart1 <- ggplot(avg_rankings, aes(x = Date, y = AverageRank, color = Discipline)) +
  geom_line(linewidth = 1.2) +
  geom_point(linewidth = 3) +
  scale_y_reverse() +
  labs(
    title = "Adam Ondra's Average IFSC Rankings (2010–2024)",
    x = "Year",
    y = "Average IFSC Rank (Lower is Better)",
    color = "Discipline"
  ) +
  theme_minimal()

ggsave("images/chart1.png", chart1, width = 8, height = 5)

Chart 1: IFSC Rankings Over Time

Let’s start with competition. Adam Ondra has competed in IFSC events since his teens, and for a while, he dominated. This chart shows his average IFSC rankings from 2010 to 2024 in two disciplines: lead and boulder.

His golden era came between 2013 and 2017. During those years, Ondra was a near-permanent fixture on the podium in lead climbing and consistently ranked in the top 5. His bouldering rankings were slightly more volatile — still strong, but not at the same level as his lead performances.

Things start to shift around 2020–2021, coinciding with the Tokyo Olympics. That dip isn’t just a blip — it reflects a broader change in his climbing focus and possibly the rise of a younger generation of climbers who specialize in indoor formats.

Code 
# Read the data quietly, drop unnamed column
ascents <- read_csv("data/adam_all_ascents.csv", 
                    show_col_types = FALSE) |>
  select(-starts_with("..."))  # removes `...5` column to silence "New names"

# Fix dmy() warning by explicitly handling parse failures
ascents <- ascents |>
  filter(Grade %in% c("9a", "9a+", "9b", "9b+", "9c")) |>
  mutate(
    Date = suppressWarnings(dmy(Date)),  # suppress parsing warnings
    Year = year(Date)
  ) |>
  filter(!is.na(Date))  # drop any bad rows to avoid ggplot/NA warnings

# Build datasets for chart
sends_per_year <- ascents |>
  group_by(Year) |>
  summarise(HardSends = n(), .groups = "drop")

avg_ifsc_rank <- filtered |>
  group_by(Date) |>
  summarise(AvgRank = mean(Rank, na.rm = TRUE), .groups = "drop") |>
  rename(Year = Date)

combined <- inner_join(sends_per_year, avg_ifsc_rank, by = "Year")

# Create chart, suppress geom_smooth formula print, suppress ggsave output
chart2 <- suppressMessages(
  ggplot(combined, aes(x = HardSends, y = AvgRank)) +
    geom_point(size = 4, color = "#377eb8") +
    geom_smooth(method = "lm", se = FALSE, linetype = "dashed", color = "gray40") +
    scale_y_reverse() +
    labs(
      title = "Outdoor Focus vs. IFSC Performance",
      x = "Number of 9a+ Outdoor Sends",
      y = "Average IFSC Ranking (Lower is Better)"
    ) +
    theme_minimal()
)

invisible(ggsave("images/chart2.png", chart2, width = 8, height = 5))

Chart 2: Outdoor Focus vs. IFSC Performance

This chart explores the tradeoff between Adam Ondra’s outdoor pursuits and his competitive performance. Specifically, it compares the number of 9a or harder routes he sent outdoors each year with his average IFSC ranking that same year.

There’s a clear pattern: when Ondra sends more hard outdoor routes, his IFSC ranking tends to suffer. This isn’t surprising — training for the World Cup circuit is incredibly specific, and time spent on real rock is time not spent in plastic bouldering gyms or lead training facilities.

The data suggests a strong inverse relationship between his outdoor activity and indoor performance. That may sound like a bad thing, but it also illustrates something important: Ondra wasn’t trying to win every competition. He prioritized a broader legacy — one that includes the world’s hardest outdoor climbs.

Code 
climbers <- tibble::tibble(
  Climber = c("Adam Ondra", "Jakob Schubert", "Stefano Ghisolfi", "Tomoa Narasaki", "Sean Bailey"),
  HardestGrade = c("9c", "9b+", "9b+", "9a+", "9b"),
  Avg_IFSC_Rank = c(5.2, 6.1, 8.3, 7.5, 9.2),
  Medals = c(38, 31, 19, 17, 10)
)

grade_map <- c("9a" = 28, "9a+" = 29, "9b" = 30, "9b+" = 31, "9c" = 32)
climbers$GradeNumeric <- grade_map[climbers$HardestGrade]

chart3 <- ggplot(climbers, aes(x = GradeNumeric, y = Avg_IFSC_Rank, size = Medals, label = Climber)) +
  geom_point(alpha = 0.7, color = "#e41a1c") +
  geom_text(vjust = -1, size = 4) +
  scale_x_continuous(breaks = 28:32, labels = names(grade_map)) +
  scale_size_continuous(range = c(4, 12)) +
  scale_y_reverse() +
  labs(
    title = "Elite Climbers: Outdoor Difficulty vs. IFSC Ranking",
    x = "Hardest Outdoor Grade",
    y = "Average IFSC Ranking (Lower is Better)",
    size = "IFSC Medals"
  ) +
  theme_minimal()

ggsave("images/chart3.png", chart3, width = 8, height = 5)

Chart 3: How Ondra Stacks Up Against Other Elite Climbers

So how does Ondra compare to other top climbers?

This chart plots five elite climbers, comparing the hardest outdoor grade they’ve sent against their average IFSC competition rank. Bubble size represents the number of IFSC medals they’ve won.

Ondra is in a league of his own. He’s the only climber in this group to have sent a confirmed 9c — the hardest grade ever climbed — and still maintain a top-tier competition record. Others, like Jakob Schubert and Stefano Ghisolfi, have similar IFSC rankings and lots of medals, but their outdoor grades top out slightly lower.

Then there’s Tomoa Narasaki, a bouldering specialist, and Sean Bailey, an American climber with impressive medals but less outdoor pedigree. They both excel in specific areas, but don’t show the same all-around dominance.

This reinforces the idea that Ondra isn’t just good — he’s versatile, and he’s stayed relevant across a much wider set of challenges than most climbers ever attempt.
